This folder contains a PDF-ready client brief for the LMS demo.

## Files
- `Client-Demo-Brief.md`: Master document to export as PDF.
- `assets/`: Place your logo and screenshots here.

## Branding
- Add your logo to `./assets/logo.png` (or update `logo_path` in the frontmatter).
- Update `primary_color` and `secondary_color` with your brand hex codes in the frontmatter.

## Fill-in Details
- Demo URL and credentials for Admin, Teacher, Student.
- Links for App URL, Docs/Notion, Support.

## Screenshots to Capture
- Admin Dashboard, Courses, Assignments
- Teacher Dashboard, Assignments (with submissions), Attendance
- Student Dashboard, Assignments (submission view), Reports

Save PNG files in `./assets/` with clear names (e.g., `admin-dashboard.png`).

## Export to PDF
1) Open `Client-Demo-Brief.md` in a Markdown viewer/editor (VS Code with Markdown Preview, Obsidian, or a docs tool that supports print to PDF).
2) Replace all placeholders and verify screenshots are available under `./assets`.
3) Print to PDF with background graphics enabled, A4/Letter portrait, normal margins.
4) Save as `LMS-Client-Demo-Brief.pdf`.

## Optional
- Add page numbers and a small footer (company name, date) in your PDF export settings if available.
